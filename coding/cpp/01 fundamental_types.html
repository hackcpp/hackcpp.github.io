数据类型(Fundamental Types)
布尔类型(Booleans)
bool b1 = true;
bool b2 = false;
字符类型(Characters)
  • 最小的整数类型,通常只有1字节
  • on x86/x86-64 values ∈ [-128,127]
    • char c = 'A';  // character literal
char a = 65;   // same as above
    有符号整数(Signed Integers)
    n bits ⇒ values ∈ [-2(n-1), 2(n-1)-1] 
    short s = 7;  
int   i = 12347;
long  l1 = -7856974990L;
long long  l2 = 89565656974990LL; 
// ' digit separator C++14
long l3 = 512'232'697'499;
    无符号整数(Unsigned Integers)
    n bits ⇒ values ∈ [0, 2n-1] 
    unsigned u1 = 12347U; 
unsigned long u2 = 123478912345UL;  
unsigned long long u3 = 123478912345ULL;  
// non-decimal literals
unsigned x = 0x4A;          // hexadecimal
unsigned  b = 0b10110101;    // binary C++14
    浮点类型
  • float: IEEE 754 32 bit
  • double: IEEE 754 64 bit
  • long double: 80-bit on x86/x86-64
    • float       f  = 1.88f;
double      d1 = 3.5e38;
long double  d2 = 3.5e38L; C++11
// ' digit separator C++14
double  d3 = 512'232'697'499.052;
    数字表示法(Number Representations)
    类型的内存大小(Memory Sizes of Types)
    所有类型的大小都是 sizeof(char) 的整数倍 
    cout << sizeof(char);   // 1
cout << sizeof(bool);   // 1
cout << sizeof(short);  // 2
cout << sizeof(int);    // 4
cout << sizeof(long);   // 8

    数值限制(std::numeric_limits)

    算数运算(Arithmetic Operations )
    Operators
  • a ⊕ b: 返回 a ⊕ b 的运算结果
  • a ⊕= b: a ⊕ b 的运算结果赋值给 a
    • int a = 4;          设置变量 a 值为 4
int b = 3;          设置变量 b 值为 3

    
a = a + b;          a: 7    add
a += b;             a: 10
a = a - b;          a: 7    subtract
a -= b;             a: 4
a = a * b;          a: 12   multiply
a *= b;             a: 36
a = a / b;          a: 12   divide
a /= b;             a: 4
a = a % b;          a: 1    remainder of division (modulo)
    自增/自减(Increment/Decrement )
  • 变量值自增/自减1
  • 前置表达式 ++x / --x 返回自增/自减后的值
  • 后置表达式 x++ / x-- 返回自增/自减前的值
    • int a = 4;    a: 4           
int b = 3;            b: 3

    
b = a++;      a: 5    b: 4
b = ++a;      a: 6    b: 6
b = --a;      a: 5    b: 5
b = a--;      a: 4    b: 5

    比较运算(Comparisons)
    2路比较(2-way Comparisons)
    比较结果为 true 或者 false
    int x = 10;                       
int y = 5;                 result  operator

    
bool b1 =  x == 5;         false   equals     
bool b2 = (x != 6);        true    not equal       
bool b3 = x > y;           true    greater     
bool b4 = x < y;           false   smaller     
bool b5 = y >= 5;          true    greater/equal    
bool b6 = x <= 30;         true    smaller/equal
    3路比较(3-Way Comparisons)
    比较 2 个对象的相对顺序 
    (a <==> b) < 0	if a < b 
(a <==> b) > 0	if a > b  
(a <==> b) == 0	if a and b are equal/equivalent

    布尔逻辑(Boolean Logic)
    操作符(Operators)
    bool a = true;
bool b = false;

    
bool c = a && b;      // false    logical AND
bool d = a || b;      // true     logical OR
bool e = !a;          // false    logical NOT

    
Alternative Spellings:
bool x = a and b;     // false
bool y = a or b;      // true
bool z = not a;       // false
    转化成bool(Conversion to bool)
  • 0 is always false;
  • everything else is true;
    • bool f = 12;   // true   (int → bool)
bool g = 0;    // false  (int → bool)
bool h = 1.2;  // true   (double → bool)

    位运算(Bitwise Operations)
    位逻辑运算(Bitwise Logic)
  • a & b bitwise AND
  • a | b bitwise OR
  • a ^ b bitwise XOR
  • ~a bitwise NOT (one's complement)
    •                                       memory bits:

    
std::uint8_t a = 6;                   0000 0110
std::uint8_t b = 0b00001011;          0000 1011

    
std::uint8_t c1 = (a & b);  // 2      0000 0010
std::uint8_t c2 = (a | b);  // 15     0000 1111
std::uint8_t c3 = (a ^ b);  // 13     0000 1101
std::uint8_t c4 = ~a;       // 249    1111 1001
std::uint8_t c5 = ~b;       // 244    1111 0100

    
// test if int is even/odd:           result:
bool a_odd  = a & 1;                  0 ⇒ false
bool a_even = !(a & 1);               1 ⇒ true
    位移运算(Bitwise Shifts)
  • x << n: 返回变量x的比特(bits)向左移动n位后的值
  • x >> n: 返回变量x的比特(bits)向右移动n位后的值
  • x <<= n: x的比特(bits)向左移动n位后的值赋值给x
  • x >>= n: x比特(bit)向右移动n位后的值赋值给x
    •                                         memory bits:
std::uint8_t a = 1;                     0000 0001

    
a <<= 6;  // 64                         0100 0000
a >>= 4;  // 4                          0000 0100

    
std::uint8_t b1 = (1 << 1);  // 2       0000 0010
std::uint8_t b2 = (1 << 2);  // 4       0000 0100
std::uint8_t b3 = (1 << 4);  // 16      0001 0000
